# Localization

There are some useful hints for UI localization teams.

## Some localization strings can include placeholders.
Example:  _Change Password for %username%_  

If you are translating this string on some language with inflective placeholder content then you need to name a placeholder.  
_Change Password for user "%username%"_    
_Сменить пароль для пользователя Иван Петров_
